# AGENTS.md

## Project

ホテルニューイタヤ デジタルサイネージCMS試作の共同作業用フォルダです。

このフォルダはGitHub公開・共同作業対象として扱い、親フォルダにある制作元データ、PDF、PPTX、PSD、AIなどは含めません。

## Scope

編集対象:

- `index.html`
- `styles.css`
- `app.js`
- `assets/`
- `README.md`
- 各種運用メモ

含めないもの:

- 親フォルダの制作元データ
- `.DS_Store`
- `_deploy/`
- 一時ファイル
- APIキーやパスワード

## Rules

- 表示確認はできるだけ `http://localhost:5173/` などのローカルサーバー経由で行う。
- TableCheckなど外部APIの認証情報はブラウザ側コードに書かない。
- 広告・会場案内の既存仕様を壊す変更は避ける。
- 変更後は、管理画面・広告①・広告②・会場案内の主要画面を確認する。

## Collaboration

共同作業時は、原則としてブランチを分けて作業します。

- `main`: CL確認用の安定版
- `feature/...`: 新機能
- `fix/...`: 表示崩れや文言修正

作業後は、変更内容、確認した画面、残る確認事項を簡潔に記録してください。

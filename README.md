# ホテルニューイタヤ デジタルサイネージCMS 試作

## 概要

50インチ縦型モニターでの表示を想定した、ブラウザベースのCMS試作です。

## 作業場所

今後の共同作業・GitHub反映は、この `itaya_cms` フォルダを本拠地にします。
旧 `CMS_prototype_20260513` は試作履歴として残し、基本的には編集しません。

- 広告利用画面①: 縦/横を選択して表示。縦は1枚表示、横は上段素材・下段素材を別々に登録して2枚上下表示
- 広告利用画面②: ①と同じ仕様で、素材・縦横設定・スライド秒数を別管理
- 広告サンプル: 縦2枚、横上段1枚、横下段1枚のサンプル画像を同梱。広告①/②それぞれサンプル復元・サンプル削除に対応
- 会場案内: 時間、会場、名前を登録し、時間帯ごとに表示。時間帯自動表示/終日すべて表示を切替可能。終日表示では終了済みイベントの表示/非表示を選択可能。会場名の前に館・階ラベルを表示。名前は改行入力と登録後の編集に対応。1ページ10件表示、11件以上は指定秒数でスライド。ライト/ダーク切替に対応
- 会場案内の現在対象強調: 開始30分前から開始後2時間までのブロックを強調表示

広告素材を複数登録した場合は、管理画面で指定した秒数ごとにスライドショー表示します。
初期値は5秒です。
PDFが複数ページの場合は、1ページずつ広告素材として扱います。
PDF表示は文字欠けを抑えるため、ブラウザ標準のPDF表示を優先します。

## 開き方

このフォルダで簡易サーバーを起動して、ブラウザで以下を開きます。

```sh
python3 -m http.server 5173
```

- 管理画面: `http://localhost:5173/`
- 広告表示①: `http://localhost:5173/index.html?screen=ad1`
- 広告表示②: `http://localhost:5173/index.html?screen=ad2`
- 会場案内: `http://localhost:5173/index.html?screen=venue`

## 保存について

この試作では、登録データは同じブラウザ内に保存されます。
本番CMS化する場合は、クラウドまたは社内サーバー側に素材・会場案内データを保存する構成に差し替える想定です。
